## Features

Bu klasör, projedeki özellikleri ve kullanılan kaynakları (API, kütüphane, araçlar) detaylandırır.

### Dosyalar

- `ai-integration.md`: Gemini/OpenAI entegrasyonu ve fallback mantığı
- `conversation-memory.md`: localStorage ile konuşma belleği
- `ui-ux.md`: chat görünümü, kart tasarımı, duyguya göre tema ve maskot
- `deployment.md`: Netlify deploy akışı ve opsiyonel Functions yaklaşımı
- `resources.md`: kullanılan araçlar / ilham kaynakları / referanslar
