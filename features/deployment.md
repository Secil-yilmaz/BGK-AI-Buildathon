## Deployment (Netlify)

## Hedef

Projeyi hızlıca yayınlamak ve paylaşmak:

- GitHub repo → Netlify bağlama
- Otomatik build + deploy

## Tipik yapılandırma

- Build command: `npm run build`
- Publish directory: `dist`

## Opsiyonel: Netlify Functions

Repo içinde `api/` ve `functions/` klasörlerinde Function örnekleri bulunur.
Prod’da API anahtarını istemci yerine sunucu tarafında tutmak için Functions iyi bir seçenektir.
